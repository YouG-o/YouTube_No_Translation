YouTube No Translation Settings

Created by YouGo

Thanks for installing! Here you can configure how the extension behaves.

Important:
For the extension to work on YouTube tabs that were already open before installation, you must reload those tabs.

Titles

Display original titles: Main video and browsing titles, notifications, channels (on main video page).

Audio Tracks

Always switch to preferred audio track.

Descriptions

If description is translated, it will be replaced with the original description.

Subtitles

Display subtitles in your preferred language if available. If not, subtitles will be disabled. Auto generated subtitles are ignored by default.

Enable Auto Generated subtitles

When enabled, if no manual subtitles are available in your preferred language, auto-generated (ASR) subtitles will be used instead, with translation if needed.

Extra Settings

YouTube Data API v3

Use YouTube Data API v3 to get original channel name & description and a fallback if other methods fails. Very reliable but requires your own API key. Your key is stored locally, only you can access it.
Get your free API key here

Cache Management

Clear cached titles and descriptions. Use this if you're experiencing incorrect content or want to start fresh.
