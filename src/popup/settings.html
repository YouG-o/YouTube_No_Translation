YouTube No Translation Settings

Thanks for installing! Here you can configure how the extension behaves.

These settings will help you keep YouTube content in its original form.

Created by YouGo

Titles

Display original titles: Main video and browsing titles, notifications, channels (on main video page).

Audio Tracks

Always switch to preferred audio track.

Descriptions

If description is translated, it will be replaced with the original description.

Subtitles

Display subtitles in your preferred language if available. If not, subtitles will be disabled. Auto generated subtitles are ignored.

Advanced Settings

YouTube Data API v3

Use YouTube Data API v3 to get original channel name & description and a fallback if other methods fails. Very reliable but requires your own API key. Your key is stored locally, only you can access it.
